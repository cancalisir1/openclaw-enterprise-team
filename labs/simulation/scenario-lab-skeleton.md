# Scenario Lab Skeleton

Based on MiroFish principles, this simulation environment is for "what-if" architectural and behavioral scenarios rather than production code.

1. **Seed Materials**: Feed PRD and ADR text.
2. **Personas**: Spin up specific lightweight personas (Attacker, Power User, Support Rep).
3. **Simulation**: Let the agents interact for N turns around the architectural choice.
4. **Synthesis**: Report agent gathers the logs and synthesizes actionable insights for the Chief Orchestrator.
